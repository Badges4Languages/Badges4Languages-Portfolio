This plugin is a complement to the Open Badges Framework plugin. It allows students to create portfolios for each languages they are studying.

== Description ==

The students who are using the Open Badges Framework platform can, with this plugin, create portfolios for each languages they learn. This portfolios contain a form with different sections :

* Listening
* Reading
* Spoken Interaction
* Spoken Production
* Writing

For each of these sections, the student can check his competencies. Depends on what the student checked, the plugin will retrieve a level (A1, A2, ...) for each sections.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire b4l-portfolio folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
